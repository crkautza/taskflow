# TaskFlow

Um sistema de lista de tarefas (CRUD) desenvolvido em JavaScript puro, com foco em lógica de programação, manipulação do DOM, eventos, filtros dinâmicos e persistência com localStorage.

## Objetivo do Projeto

O principal objetivo do TaskFlow foi consolidar conceitos fundamentais de desenvolvimento front-end com JavaScript puro, sem o uso de frameworks ou bibliotecas externas. O foco foi:

- Manipulação avançada do DOM
- Gerenciamento de estado com objetos JavaScript
- Implementação de CRUD completo (Criar, Ler, Atualizar, Deletar)
- Uso de eventos para interatividade
- Filtros dinâmicos (Todas, Ativas, Concluídas)
- Persistência de dados com localStorage
- Organização de código com funções reutilizáveis

> **Nota:** Este projeto não foi desenvolvido com foco em responsividade para dispositivos móveis. O layout é otimizado para desktop, sendo este um ponto de melhoria para versões futuras.

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas (checkbox + texto riscado)
- Editar tarefas existentes (com opção de cancelar)
- Remover tarefas
- Filtros: Todas, Ativas, Concluídas
- Contador de tarefas (total, ativas, concluídas)
- Persistência com localStorage
- Botões desabilitados dinamicamente conforme o contexto
- Mensagem amigável quando não há tarefas no filtro

## Tecnologias utilizadas

- HTML5
- CSS3 (Flexbox, design customizado)
- JavaScript (puro, sem frameworks)
- localStorage (persistência de dados)
